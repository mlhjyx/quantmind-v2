# Phase 2.3 前置：策略市值暴露诊断

**日期**: 2026-04-11
**目的**: 搞清策略到底在选什么市值区间的股票，以及这对所有历史研究结论的影响

## 6个核心问题的回答

### Q1: 策略是微盘/小盘/中盘策略？

**结论：无SN = 纯微盘策略，SN b=0.50 = barbell大盘+微盘策略**

| 指标 | 无SN | SN b=0.50 |
|------|------|-----------|
| Top-20 平均市值 | 130.3 亿 | 5,821 亿 |
| Top-20 中位数市值 | 32.3 亿 | 3,593 亿 |
| 微盘(<100亿)占比 | **91.5%** | 42.0% |
| 小盘(100-300亿)占比 | 6.4% | **0.0%** |
| 中盘(300-500亿)占比 | 0.6% | **0.0%** |
| 大盘(>500亿)占比 | 1.5% | **58.0%** |
| 全A分位(2020/2022/2024) | — | 99.8% / 99.7% / 96.0% |

**关键发现**：SN 不是"微调"而是"翻转"——将策略从 P<25% 微盘直接推到 P>96% 超大盘，中间的小盘和中盘占比为 0%。这种 barbell 结构说明 SN 的线性调整过于粗暴。

### Q2: CSI300 基准是否合理？应用什么？

**结论：CSI300 是最差基准（corr=0.077），但基准选择不影响因子 IC**

| 基准 | 与策略相关性 | 超额Sharpe |
|------|------------|-----------|
| 全A等权 | **0.334** (最高) | 0.524 |
| CSI2000 | 0.109 | 0.513 |
| CSI1000 | 0.107 | 0.533 |
| CSI500 | 0.105 | 0.531 |
| CSI300 | 0.077 (最低) | 0.549 |

推荐基准：**全A等权**（相关性最高，但 0.33 仍很低——策略极其独特）。

**但重要发现**：Step 3 证明 IC 对基准选择完全不敏感（5 因子 delta ≈ 0）。原因：Spearman Rank IC 只看排名，减去基准日收益（所有股票相同的常数）不改变截面排名。所以"CSI300 放大 IC"的担心 **不成立**。

### Q3: CORE5 IC 有多少来自真 alpha vs 风格溢价？

**结论：因子是真 alpha，但微盘放大了 IC 约 3-4 倍**

| 因子 | 微小盘IC | 中盘IC | 大盘IC | 衰减(微→大) |
|------|---------|-------|-------|------------|
| turnover_mean_20 | -0.105 | -0.081 | -0.023 | -78% |
| volatility_20 | -0.100 | -0.083 | -0.028 | -72% |
| reversal_20 | 0.063 | 0.045 | 0.013 | -79% |
| amihud_20 | 0.030 | **0.007** | 0.014 | -53% (非单调) |
| bp_ratio | 0.066 | 0.057 | 0.028 | -58% |

- **所有因子在所有市值分组中 t > 2.5** → 跨市值有效，不是纯粹的微盘溢价
- IC 梯度明显：微盘 IC 约为大盘的 3-4 倍
- **amihud 非单调**：中盘 IC = 0.007（t=3.0，勉强显著），流动性溢价只在极端分位有效
- **bp_ratio 最稳健**：跨市值衰减最小（-58%），大盘 IC 仍有 0.028

### Q4: SN(b=0.50) 是否有效控制市值暴露？

**结论：SN 有效但过度矫正，barbell 结构非最优**

- SN 将平均市值从 130 亿推到 5,821 亿（+4367%）
- 但产生 barbell：42% 微盘 + 0% 小中盘 + 58% 大盘
- 2020/2022 年策略处于全 A P99.8%——极端大盘
- 2024 年降到 P96%，说明 SN 效果随时间波动

**隐含问题**：SN 的线性调整 `score - 0.50 × zscore(ln_mcap)` 在因子分数与市值高度相关时，会导致大小市值的翻转而非渐进平移。

### Q5: 流动性是否支撑 100 万执行？

**结论：SN 后流动性充裕，无 SN 偏低但基本可执行**

| 指标 | 无SN | SN b=0.50 |
|------|------|-----------|
| 平均日均成交额 | 4,642 万 | 148,708 万 |
| 最低日均成交额(均值) | 747 万 | 3,700 万 |

- SN 后：100 万 / 3,700 万 = 日成交 2.7% → ✅ 无冲击成本风险
- 无 SN：100 万 / 747 万 = 日成交 13.4% → ⚠️ 个别股可能有冲击成本

### Q6: 对后续实验的建议

1. **收窄 universe 到中盘（100-500 亿）**：所有 CORE5 因子中盘 IC 仍显著（t = 3.0 ~ 29.5），可避免微盘执行风险
2. **考虑用 universe filter 替代 SN**：直接限制市值范围比 SN 线性调整更可控，避免 barbell 问题
3. **amihud 在中盘几乎无效（IC=0.007）**：如收窄到中盘，amihud 可能需要替换
4. **基准改用全 A 等权或 CSI1000**：与策略实际暴露更匹配
5. **SN beta 需重新调参**：当前 b=0.50 过度矫正，如保留 SN 应测试 b=0.15~0.30

### Spearman Rank IC 基准无关性
- IC = rank_corr(factor_values, stock_return - benchmark_return)
- 减去同一天所有股票相同的常数不改变排名
- 因此 IC(vs CSI300) = IC(vs 全A等权) = IC(vs any benchmark) — 数学必然
- **重要推论**：所有历史 IC 研究结论不受基准选择影响
