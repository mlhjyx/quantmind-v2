# 发现: 等权框架因子扩展的稀释效应
- 日期: 2026-04-08
- 证据: 9个独立候选因子(|corr|<0.7 vs Active)逐一加入5因子等权组合，全部Sharpe下降(0.84→0.52~0.73)
- 根因: 等权框架下第6因子稀释原5因子信号。新因子IC独立性高但边际alpha不足以补偿稀释
- 启示: 因子扩展不应在等权框架内逐一添加，应考虑: (1)IC加权替代等权 (2)分层策略 (3)ML组合 (4)Modifier方式引入(不改选股)
- 关联: CLAUDE.md已知失败"IC加权/Lasso等下游优化：因子信息量不够时优化下游无效"
- 结论: NorthboundModifier(不改选股,改仓位)是比因子扩展更有效的alpha增强路径
