## 课题1：A股多因子2024-2025实证——哪些因子还有效

### 研究发现

2024-2025年A股因子表现出明显的分化：

**持续有效的因子类别**：
- 短期反转：依然是A股最稳定的Alpha来源，散户占比62%支撑反转效应
- 换手率/流动性类：换手率波动率、非流动性(Amihud)、波峰因子——长期IC靠前，月度胜率>60%
- 特质波动率(IVOL)：方向为负（低特质波动率跑赢），在中证1000中RankIC达-10.45%，但需要做信息含量重构才能用好
- 留存筹码比率（华西证券2025新研究）：IC稳定在5-7%，利用散户处置效应（盈利早卖、亏损死扛），年化超额11.78%

**2024年衰减/失效的因子**：
- 传统低频基本面因子（PE/PB/ROE）：2022年下半年起超额收益持续走平，沪深300尤其明显
- 价值因子(HML)：2024年AI/科技主题行情下严重失效——大盘成长股跑赢小盘价值股
- 动量因子：A股动量效应本身就弱于美股，2024年更是被反转碾压

**2025年Q1最新趋势（摩根量化多因子基金季报）**：
- 高频量价因子和成长类因子有明确超额
- 传统低频基本面因子面临回撤压力
- 市场延续小盘成长风格——北证50涨22.48%，沪深300跌1.21%

**关键学术发现**：
- 等权合成优于IC加权（这与我们三轮讨论的结论一致）——年化28.44%，多空超额54.4%，月度胜率>90%
- 因子在小市值宽基中表现更好——中证1000/国证2000的RankIC比沪深300高30-50%
- 因子半衰期约2个月（间隔2期IC衰减50%）——支持月度调仓频率

### 对QuantMind的建议

**现有5因子的诊断**：
我们的5因子（turnover_mean_20/volatility_20/reversal_20/amihud_20/bp_ratio）：
- turnover_mean_20 ✅ 属于持续有效的换手率/流动性类
- volatility_20 ✅ 波动类依然有效
- reversal_20 ✅ 短期反转是A股最强因子
- amihud_20 ✅ 非流动性持续有效
- bp_ratio ⚠️ 价值因子2024年严重失效——这是2024年集体失效的根源

**新因子候选（替代被否决的GPA，按优先级排序）**：
1. **特质波动率(IVOL)**：日频可算（Fama-French残差波动率），IC稳定，与现有因子正交。用现有数据可以做：回归market+size+value三因子→取残差→20日标准差。Sprint 1.3可实现
2. **留存筹码比率**：需要分钟级数据（换手率衰减模型），我们当前只有日频数据做不了。记入Phase 2（日内数据）
3. **成交额占比熵**：需要分钟级数据，同上
4. **收益率偏度(Skewness)**：20日收益率偏度，日频可算，学术有验证（Boyer et al. 2010）

**立刻让factor做的事**：
IVOL因子验证——用现有klines_daily+index_daily数据，算Fama-French三因子残差的20日标准差，方向为负（低IVOL跑赢），跑IC。这个比GPA可靠得多——有大量A股学术验证。

## 课题2：动态风控 vs 静态熔断

### 研究发现

我们当前的L1-L4是**固定阈值静态熔断**（日亏>3%/5%/月>10%/累计>25%）。学术界和业界有更智能的方法：

**静态熔断的问题**：
- 牛市高波动期频繁误触发（A股日波动2-3%是常态，L1=3%会在正常行情中频繁触发）
- 熊市低波动期反应太慢（市场已经跌了15%但每天只跌1%，L1/L2不触发）
- 不区分市场系统性下跌和策略特异性风险

**GARCH动态阈值**：
用GARCH(1,1)预测下一期波动率σ，阈值设为k×σ而不是固定数字：
- L1触发 = 日亏 > 2σ（而不是固定3%）
- 高波动期σ=3%→阈值6%，不会频繁误触发
- 低波动期σ=1%→阈值2%，能更快检测到异常

**Regime-Switching（Hamilton 1989）**：
市场有牛/熊/震荡三个regime，每个regime有不同的均值和波动率。用Hidden Markov Model检测regime转换，在regime切换时调整仓位：
- 检测到从牛市→熊市转换 → 主动降仓（不等亏损触发熔断）
- 这比"亏了再降"高级——它是"预测到要亏了提前降"

**CVaR/Expected Shortfall**：
比VaR更好的尾部风险度量——关注的是"亏超过阈值时平均亏多少"而不是"亏超过阈值的概率"。更适合做风控决策。

### 对QuantMind的建议

**Phase 0-1A：保持静态L1-L4**，但调整阈值使其更合理（你已确认L1=3%/L2=5%/L3=10%/L4=25%）。加一个简单改进：用20日滚动波动率做阈值自适应——当市场波动率翻倍时，阈值也翻倍。这不需要GARCH，一个简单的乘法就行。

**Phase 1B-1C：引入GARCH动态阈值**，替代固定数字。用arch库（Python），已经在pyproject.toml范围内。

**Phase 3：引入Regime-Switching**，主动检测市场状态切换做预防性降仓。

**不建议现在做的**：强化学习风控——学术上有论文但实际落地案例极少，风险太大不适合管钱。

## 课题3：Paper Trading衰减预算——实盘到底衰减多少

### 研究发现

这是最关键的课题。strategy提出"衰减15-25%"，现在有学术数据了：

**学术证据（多个来源汇总）**：
- Research Affiliates：实盘因子收益是纸面收益的**50%或更低**——价值和市场因子收益"被减半或更差"，动量因子"完全没有给终端投资者带来收益"
- 因子衰减研究：样本外平均衰减50%，做了规模调整后衰减25-50%
- McLean & Pontiff (2016)：因子发表后样本外衰减约32%
- Alpha Decay论文：月度衰减信号的交易成本可以减少总收益的50%以上
- 因子半衰期：价值因子25个月，低波动12个月，动量8个月

**但这些是大基金的数据，不完全适用于我们**：
- 大基金规模大，市场冲击严重→我们100万规模冲击可忽略
- 大基金做多空对冲，空头执行成本高→我们只做多头
- 大基金多策略竞争激烈→我们的Alpha来源（A股散户行为）竞争相对低

**更适合我们的估计**：
- 交易成本衰减：佣金万1.5+印花税0.05%+滑点约0.1%/单边≈月度调仓0.3-0.5%年化衰减 → Sharpe衰减约5-10%
- 执行延迟衰减：T日信号→T+1执行，overnight gap风险 → 再衰减5-10%
- 因子拥挤衰减：100万规模几乎无拥挤 → 0%
- 模型不确定性/过拟合：Bootstrap CI [0.41, 2.19]本身已涵盖

**综合估计：Sharpe从回测到实盘衰减15-25%是合理的**。
- 回测Sharpe 1.29 → 实盘预期 0.97-1.10
- Paper Trading毕业标准0.90 → 已经预留了30%衰减空间，足够

### 对QuantMind的建议

**strategy的15-25%估计是准确的**，有学术支撑。但要区分衰减来源：
- 可控衰减（交易成本、执行延迟）：通过优化执行算法可以减少
- 不可控衰减（因子过拟合、市场变化）：只能通过因子分散化对冲

**Paper Trading期间的核心观测**：
- 如果Paper Trading Sharpe > 1.0 → 实盘有望保持>0.75，可以转实盘
- 如果Paper Trading Sharpe 0.7-1.0 → 勉强，需要评估原因
- 如果Paper Trading Sharpe < 0.7 → 不应该转实盘，需要先改进策略

**毕业标准0.90是合适的**——给了30%衰减空间，学术数据支持。

## 课题4：GP因子挖掘A股案例——DEAP怎么配

### 研究发现

GP遗传编程在因子挖掘中的应用：

**GP的核心优势**：
- 搜索空间巨大——表达式树可以组合出人类想不到的因子
- 不需要人工假设——纯数据驱动
- 可解释——输出是数学表达式，不是黑箱

**GP的核心风险**：
- 过拟合——GP非常擅长找到样本内完美的因子，但样本外失效
- 复杂度爆炸——不限制深度会生成不可解释的怪物表达式
- 计算量大——种群×代数×因子评估（每次评估需要跑截面IC）

**最佳实践（综合Qlib/RD-Agent/学术文献）**：

算子集（A股）：
- 时序算子：ts_mean, ts_std, ts_rank, ts_delay, ts_delta, ts_corr, ts_max, ts_min
- 截面算子：cs_rank, cs_zscore
- 数学算子：add, sub, mul, div(safe), abs, log, sign, power
- 不要用：复杂三角函数、矩阵运算——增加复杂度但不增加Alpha

DEAP配置建议：
```
population_size: 500        # 种群大小（太小搜索不充分，太大计算慢）
generations: 50             # 代数（早停：连续5代最优个体不变→停止）
max_depth: 4                # 树最大深度（A股实证表明4层以上IC不增加但过拟合风险飙升）
tournament_size: 5          # 锦标赛选择
crossover_prob: 0.7         # 交叉概率
mutation_prob: 0.2          # 变异概率
elitism: 10                 # 精英保留数

# 适应度函数（关键）
fitness = IC_mean × 0.4 + ICIR × 0.3 + monotonicity × 0.2 - complexity_penalty × 0.1
# complexity_penalty = tree_depth / max_depth（惩罚过深的表达式）
```

防过拟合：
- **分年验证**：fitness用2021-2023计算，2024纯OOS验证
- **复杂度惩罚**：表达式深度>3就开始扣分
- **相关性去重**：每代淘汰与已有因子corr>0.7的个体
- **Bootstrap CI**：最终候选因子CI下界>0才入库

### 对QuantMind的建议

**Sprint 1.6用DEAP时的配置**直接用上面的参数作为起点。关键是**适应度函数里一定要有复杂度惩罚和相关性检查**——没有这两项GP会生成一堆互相雷同的过拟合因子。

**与LLM因子挖掘的互补关系**：
- GP擅长：无假设搜索、数学组合、量价因子
- LLM擅长：有经济学假设、跨数据源、基本面因子
- 两者互补不替代——GP搜到的好因子让LLM分析经济学含义，LLM提出的假设让GP搜索最优表达式

## 课题5：Alpha158在A股的真实IC——哪些值得导入

### 研究发现

Qlib Alpha158是微软设计的A股因子集，官方benchmark：
- LightGBM在Alpha158上：IC=0.0399，ICIR=0.4065，年化12.84%，MDD -6.35%

Alpha158的158个因子分为几大类：
- **KBAR类（K线形态，约20个）**：(close-open)/open, (high-low)/close, (close-low)/(high-low)等。我们完全没有这类因子——**纯增量**
- **价格类（约30个）**：close/mean(close,5), close/max(high,20)等。与我们的momentum/reversal部分重叠
- **量类（约20个）**：volume/mean(volume,5), volume变化率等。与turnover_mean_20部分重叠
- **量价交叉类（约30个）**：corr(close,volume,10), corr(close,log(volume),10)等
- **滚动统计类（约30个）**：std(close,5/10/20), ts_rank(close,5/10/20)等
- **技术指标衍生类（约28个）**：基于RSI/MACD/KDJ的衍生特征

**根据2024-2025实证，最值得导入的子集**：
1. KBAR类（全部20个）——我们没有，纯增量，用OHLC就能算
2. 量价交叉类中的corr系列——补充量价相关性维度
3. 滚动统计中的ts_rank系列——截面排名因子，与我们的因子池互补
4. 不需要导入的：简单价格均线比（close/SMA）——与reversal高度重叠

### 对QuantMind的建议

**Sprint 1.3因子扩展的具体执行方案**：

第一批（纯增量，立刻做）：
```
KBAR因子（20个）：
  kbar_open_close = (close - open) / open        # 实体大小
  kbar_high_low = (high - low) / close            # 振幅
  kbar_close_low = (close - low) / (high - low)   # 上影线比
  kbar_open_low = (open - low) / (high - low)     # 下影线比
  ... 共20个变体（不同窗口期5/10/20）
```

第二批（补充维度）：
```
量价相关（6个）：
  corr(close, volume, 5/10/20)
  corr(close, log(volume+1), 5/10/20)

截面排名（6个）：
  ts_rank(close, 5/10/20)
  ts_rank(volume, 5/10/20)
```

**全部用暴力枚举管道批量跑IC，Gate筛选（IC>0.02, corr<0.7, 覆盖>1000）**。
预计从32个候选中通过15-20个独立因子。
